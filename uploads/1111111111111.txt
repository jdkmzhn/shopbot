CCM19 – Cookie Consent Manager / Tool

Cookie Consent rechtskonform einholen!

Unser Tool bietet effizientes Consent Management, so sind Sie immer komplett auf der sicheren Seite bei sämtlichen rechtlichen Anforderungen (BGH, EuGH, DSGVO, TDDDG) an die technische Umsetzung eines Cookie Banners.

Passend & nahtlos integrieren!

Optimale Nutzererfahrung durch komplette Anpassung an Ihr Design/CI. So erzielen Sie die höchstmöglichen OptIn Raten.

Updates inklusive!

Wenn sich an rechtlichen Rahmenbedingungen etwas ändert, bekommen Sie notwendige Anpassungen frei Haus. Sie müssen die rechtlichen Rahmenbedingungen nicht mehr im Auge behalten.

Alle Daten in Deutschland!

Die gesamte Infrastruktur inkl. aller Server ist komplett in Deutschland gehostet – es werden keine Daten in ein unsicheres Drittland (z. B. die USA) exportiert. Dies gewährleistet ein sicheres und zuverlässiges Consent Management, bei dem alle Daten ausschließlich innerhalb Deutschlands verwaltet werden.

Durchdachte Cookie Consent Spezialversion für Agenturen und Provider 
NoLabel Cookie Consent

Agenturen und Hostingprovidern bieten wir eine spezielle NoLabel fähige Version (Agency/Hosting Version), die Sie unter Ihrer eigenen Marke vertreiben können.

Diese Version bietet auch eine Headless API, um mit externen Systemen Nutzer in CCM19 zu erzeugen – so können Sie das System optimal mit Ihrem Buchungssystem verbinden.

Mandantenfähig

Die Agency/Hosting Version des CMP CCM19 ist mandantenfähig, d. h. Sie können mit einer Installation beliebig viele Kunden oder eigene Websites verwalten.

Jeder Kunde hat seinen eigenen Login. Hier kann er seine Cookie Daten verarbeiten – mit beliebig vielen Domains.

Faire Preise 

Abgerechnet wird die CCM19 Agency Version pro Domain / pro Monat. Der Preis ist natürlich auch nach Menge staffelbar.

Cookie Consent Tool als Download Variante

Ihre Besucher – Ihre Daten – alles auf Ihrem Server.

Die komplette Software läuft hier auf Ihrem Server oder Ihrem Account bei Ihrem Provider. Je nach Traffic sind die technischen Anforderungen minimal.

Alle Daten über alle Besuche(r) bleiben in der Download Variante vollständig unter Ihrer Kontrolle. Es werden keine Daten an externe Server oder Dienste übertragen.

Viele andereCookie Consent Tools binden Skripte von externen Servern ein und übermitteln auch Daten dorthin. Mit der CCM19 Cookie Consent Tool Downloadvariante sind Sie völlig unabhängig und behalten die 100%ige Kontrolle über Ihre Daten und die Ihrer Besucher. Ein Zugriff durch Dritte ist damit ausgeschlossen.

Die CCM19 Cookie Consent Manager Download Version ist in allen  Tarifen ab dem kleinen Starter Tarif buchbar.

Falls Sie keine Installation auf Ihrem Webserver durchführen können oder wollen, nutzen Sie einfach die normale Service bzw. Cloud Variante an.

Im CCM19 Cookie Consent Tool wird die Einwilligung  protokolliert

Die DSGVO fordert, die Einwilligung jedes Nutzers zu protokollieren (Art. 7 Abs. 1 DSGVO). Unser Cookie Consent Manager ist die perfekte Lösung für alle, die sicherstellen möchten, dass ihre Website diesen Anforderungen der DSGVO entspricht. Mit unserem Tool können Sie die Einwilligung jedes Nutzers protokollieren und dokumentieren - ganz einfach und unkompliziert.

Ob Sie einen Online-Shop betreiben oder eine informative Webseite haben, der Cookie Consent Manager ist ein Muss für jeden, der sich an die Datenschutzvorschriften halten möchte.

Die Zustimmungshistorie ist jederzeit für Administratoren einsehbar. Setzen Sie auf unseren Cookie Consent Manager und sorgen Sie dafür, dass Ihre Website datenschutzkonform ist - Ihre Besucher werden es Ihnen danken!

FAQ zum Cookie Consent Manager
Gibt es einen Installationsservice?

Ja – den Installationsservice können Sie bequem im Warenkorb buchen. Klicken Sie dazu in der Preistabelle auf das gewünschte Produkt. Danach werden Sie zum Warenkorb weitergeleitet, wo Sie den Installationsservice dazu buchen können.

Ist CCM19 für jedes System verfügbar?

CCM19 ist grundsätzlich für jedes System verfügbar und funktioniert in mehr als 99,99% der Fälle technisch out of the box solange der CCM19 Code Schnipsel korrekt eingebunden werden kann. (Man muss ja auch ins Auto einsteigen können um es zu fahren.)

 

Gibt es eine Online Demo Version?

Sie können sich die Online Demo Version hier anschauen – diese ist voll funktionsfähig und wird alle 30 Minuten auf den Ausgangszustand zurückgesetzt. Hier gehts zur CCM19 Demoseite.

 

 

Mindestlaufzeit und Kündigungsfrist ?

In den Tarifen gibt es keine lange Laufzeiten und somit auch keine lange Kündigungsfristen. Die Laufzeit beträgt immer ein Monat / Jahr (je nach Wunsch und Auswahl) und ist jederzeit zum Ende der Laufzeit von beiden Seiten aus kündbar. Fairer gehts eigentlich nicht.

Bleiben alle Daten unter meiner Kontrolle?

Ja! Die Downloadvariante von CCM19 bietet Ihnen diese Möglichkeit und übermittelt aus diesem Grund keinen einzigen Datensatz Ihrer Besucher an irgendeinen externen Server oder Dienst.

In der Cloud Service Variante werden die anonymisierten Daten an unsere Server übertragen. Es findet keinerlei Auswertung/Verarbeitung personenbezogener Daten statt.

Cookie Consent Tool Systemanforderungen

Die Cloud Version hat keine speziellen Systemanforderungen!

Die Self Service/Download Version 
braucht lediglich PHP ab 7.2 und das Modul mod_rewrite auf Ihrem LAMP Server, um zu funktionieren.
Ebenfalls ist NGINX möglich – eine Konfigurationsdatei liegt im Download bei. 

Ergo: Überall, wo ein CMS oder Shopsystem läuft, läuft auch CCM19.

Eine Datenbank ist nicht notwendig. CCM19 ist kompatibel mit allen gängigen CMS und Shopsystemen und kann überall integriert werden.
Die Service Tarife erfordern lediglich die Möglichkeit, ein Javascript in Ihre Site einzubinden – eine Installation ist nicht notwendig!
Sie müssen lediglich darauf achten das das CCM19 Skript als erstes in die Seite integriert werden kann.
Nahezu alle Anbieter ermöglichen das.

Auftrag und Angebot
Unser Angebot: Medien und Beratung für Schule und Bildung

Das Kreismedienzentrum unterstützt die Bildungsarbeit mit und durch Medien, insbesondere von Personen, die mit Kindern und Jugendlichen im Stadt- und Landkreis Heilbronn arbeiten, mit einem vielfältigen Angebot:

Unterrichtsmedien online und zum Ausleihen: Streaming und Download von schulbezogenen Onlinemedien; Verleih von didaktischen DVDs, Medienkoffern etc.
Medientechnik: Verleih von medientechnischen Geräten zur Präsentation und praktischen Medienarbeit (z. B. iPads, Beamer, Leinwände, Videokameras u.v.m.)
Videokonferenz-Service: Datenschutzkonformes Angebot für die Nutzung im schulischen Rahmen (z. B. Fernunterricht) auf Basis von Jitsi Meet
Beratung und Schulung im Bereich audiovisuelle digitale Medien - für Pädagogen und andere Interessierte
Schulnetzberatung: Fachkundige Begleitung bei der Planung, Ausschreibung und Instandhaltung von schulischen Computernetzen
Medienpädagogische Beratung: Begleitung von schulischen Medienprojekten und Beratung in Fragen der Medienentwicklungsplanung und des Medieneinsatzes
Ohrenspitzer-Stützpunkt: Beratung, Workshops und Anleitung durch die Ohrenspitzer-Referentin
Projektleitung und Redaktion des Projekts "Mein-Heilbronn.org - Entdecke deine Stadt" in Kooperation mit dem Staatlichen Schulamt Heilbronn (Gesamtleitung: Stadtarchiv Heilbronn)

Dabei unterstützt der Landkreis Heilbronn nicht nur die Medienarbeit der öffentlichen Schulen und Hochschulen, sondern auch die der Kindergärten und außerschulischen Kinder- und Jugendarbeit sowie der Volkshochschulen und Musikschulen (siehe Nutzungsbedingungen).

Der Kompetenzverbund der Medienzentren (Medienzentrenverbund)

Stand: 12.11.2020 (FL)

Das Kreismedienzentrum Heilbronn ist Teil des Kompetenzverbunds der Medienzentren Baden-Württembergs. Dieser Verbund aus Kreis-, Stadt- und zwei Landesmedienzentren entwickelt das Aufgabenspektrum der Medienzentren ständig weiter und passt es an aktuelle Entwicklungen an.

Gleichzeitig werden die jeweils vor Ort herrschenden Bedingungen berücksichtigt und regionale Schwerpunkte gesetzt.

Die Leiterinnen und Leiter der Medienzentren sind im Landesarbeitskreis Medienzentren Baden-Württemberg, dem LAK Medien, sowie in einer Arbeitsgruppe Medienzentren des Landkreistags Baden-Württemberg organisiert.

Trägerschaft

Träger des Kreismedienzentrums ist der Landkreis Heilbronn, der in der Finanzierung von der Stadt Heilbronn im Rahmen einer öffentlich-rechtlichen Vereinbarung unterstützt wird. Das Kreismedienzentrum ist dem Schul- und Kulturamt des Landkreises zugeordnet.

Medienzentrengesetz

Die Grundlage unserer Arbeit ist im baden-württembergischen Medienzentrengesetz aus dem Jahre 2001 beschrieben, das die Aufgaben des Landesmedienzentrums sowie der Kreismedienzentren regelt (Auszüge):
Gesetz über die Medienzentren (Medienzentrengesetz) vom 1. Februar 2001 (Drucksache 12 / 5981)
§ 1
Das Landesmedienzentrum Baden-Württemberg sowie die Stadt- und Kreismedienzentren haben die Aufgaben zu erfüllen, die sich aus der Verwendung audiovisueller und digitaler Medien in der Erziehungs- und Bildungsarbeit der öffentlichen Schulen ergeben.
§ 11 Aufgaben der Stadt- und Kreismedienzentren
(1) Die Landkreise und die Stadtkreise unterhalten Kreis- und Stadtmedienzentren. Diese beschaffen für die Schulen erforderliche audiovisuelle und digitale Medien, stellen diese bereit und erfüllen mit diesen Medien verbundene pädagogische und organisatorische Aufgaben. Sie können bei der Unterstützung und Beratung im Bereich Multimediatechnik an Schulen einschließlich pädagogischer Netzwerke mitwirken (Support).
(2) Das Landesmedienzentrum berät die Stadt- und Landkreise als Träger der Stadt- und Kreismedienzentren bei der Einrichtung der Medienzentren und der Wahrnehmung der damit verbundenen Aufgaben, es koordiniert und unterstützt die Arbeit der Stadt- und Kreismedienzentren einschließlich der erforderlichen Fachfortbildungen des pädagogischen Personals der Medienzentren und führt in Zusammenarbeit mit ihnen Fortbildungsveranstaltungen durch.

Während des Aufrufs unserer Website werden automatisch Informationen an den Server unserer Website gesendet, um eine fehlerfreie Bereitstellung unserer Website gewährleisten zu können. Folgende Daten werden hierbei erhoben:

Nehmen Sie mit uns Kontakt auf, willigen Sie in die Verarbeitung Ihrer Daten zur Bearbeitung Ihrer Anfrage, sowie für weitere Korrespondenz, ein.

Eine Weitergabe der Daten an Dritte oder eine sonstige Auswertung findet nicht statt. Hiervon unberührt bleiben zwingende gesetzliche Bestimmungen, insbesondere gesetzliche Aufbewahrungsfristen.

2.    Speicherdauer
Ihre personenbezogenen Daten werden gelöscht oder gesperrt, sobald der Zweck der Speicherung entfällt. Zwingende gesetzliche Bestimmungen, insbesondere gesetzliche Aufbewahrungsfristen, bleiben hiervon unberührt.

3.    Cookies
Zusätzlich zu den zuvor genannten Daten werden bei Ihrer Nutzung unserer Website Cookies auf Ihrem Rechner gespeichert. Bei Cookies handelt es sich um kleine Textdateien, die auf Ihrer Festplatte dem von Ihnen verwendeten Browser zugeordnet gespeichert werden und durch welche der Stelle, die das Cookie setzt (hier durch uns), bestimmte Informationen zufließen. Cookies können keine Programme ausführen oder Viren auf Ihren Computer übertragen.

Durch die Verwendung der Cookies kann Ihr Computer als technische Einheit während Ihres Besuches auf der Website identifiziert werden. So wird Ihnen die Verwendung unseres Angebots, auch bei einem Seitenwechsel, erleichtert und unsere Website nutzerfreundlicher.

Sie können in der Regel Ihren Internetbrowser so einstellen, dass Sie über das Setzen von Cookies informiert werden und Cookies im Einzelfall erlauben oder ausschließen können und bereits vorhandene Cookies löschen können. Die Deaktivierung der Cookies kann zu einer Beeinträchtigung der Funktionalität der Website führen.

Fernunterricht
Für die Organisation und Durchführung des Fernunterrichts steht allen Lehrkräften in Baden-Württemberg das Landesangebot Moodle zur Verfügung. Es beinhaltet den Zugriff auf das Videokonferenz-Tool "Big Blue Button".
Für datenschutzsichere Videokonferenzen steht darüber hinaus allen Lehrkräften im Stadt- und Landkreis das Angebot "Jitsi Meet Heilbronn" zur Verfügung.
Für das übersichtliche und niederschwellige Strukturieren von Unterrichtsinhalten ähnlich dem Angebot "Padlet" bietet sich derzeit das deutsche, nach eigenen Angaben DS-GVO-konforme Angebot "TaskCards.de" an. Lehrkräfte im Landkreis Heilbronn können den Dienst nach Anmeldung über uns kostenlos nutzen.
Office 365 und Datenschutz

Office 365 von Microsoft mit seinen vielfältigen, auch kollaborativen Funktionen ist an vielen Schulen beliebt und wird eingesetzt. Angesichts der Pandemienotlage wird die Software aus Sicht der Kultusverwaltung derzeit geduldet, obwohl Datenschutzbedenken bestehen.

Nach einer Prüfung einer speziell konfigurierten Version von Office 365 als Teil der neuen Bildungsplattform für Schulen rät der Landesbeauftragte für Datenschutz und Informationsfreiheit Baden-Württemberg (LfDI) von der Nutzung dieser Software ab (Pressemitteilung vom 7. Mai 2021).

Auch wenn sich die Empfehlung auf eine bestimmte Version von Office 365 bezieht, kann man vermuten, dass auch von anderen Versionen abgeraten werden wird. Wir empfehlen daher, bei der Neueinführung dieses Produkts zur Verwendung im Unterricht zurückhaltend zu sein, bis die Sachlage geklärt ist. Schulen, die es bereits einsetzen, werden weitere Entscheidungen abwarten müssen.

Update: Im Schreiben der baden-württembergischen Kultusministerin Schopper an alle Schulen vom 22. Juli 2021 wird zugesichert, dass den Schulen, die "derzeit Microsoft-Produkte einsetzen", diese Produkte nicht pauschal vom LfDI untersagt werden, "bis das Land eine datenschutzkonforme Lösung gesamtheitlich zur Verfügung stellt."

Inwieweit diese pauschale 'Einigung' zwischen Kultusministerium und Datenschützer auch auf mögliche Einzelfälle, also einzelne Beschwerden gegen den Einsatz der Microsoft-Produkte, anwendbar sein wird, ist aus meiner Sicht ungeklärt. (FL)

Datenschutz

Umfangreiche Informationssammlung zum Thema Datenschutz in Schulen, auf die sich Lehrkräfte in ihrem dienstlichen Handeln berufen können, findet sich auf dem Fortbildungsserver für Lehrkräfte Baden-Württembergs: lehrerfortbildung-bw.de/sueb/recht/ds_neu/

Außerdem stellt das Referat 15 der Kultusverwaltung Baden-Württembergs eine Infosammlung, u. a. mit den "FAQ Datenschutz an Schulen" bereit: www.it.kultus-bw.de

TIPP: Da die an beiden Orten einsehbaren FAQs ständig aktualisiert werden, empfiehlt es sich, stets aktuell hier nachzusehen, anstatt sich auf evtl. veraltete Unterlagen zu verlassen.

Personenbezogene Daten verschlüsseln

Jeder, der personenbezogene oder behördenkritische Daten auf PCs oder Mobilgeräten verarbeitet oder auf mobilen Datenträgern (z. B. USB-Stick) speichert, ist verpflichtet, ein Verschlüsselungsprogramm zu nutzen. Ohne das richtige Kennwort sind die Daten für Fremde unleserlich, und deren Sicherheit ist somit auch bei Verlust des Datenträgers gewährleistet.

Von der Kultusverwaltung Baden-Württemberg wird die Software Veracrypt empfohlen. Download von heise online: http://www.heise.de/download/veracrypt-1195747.html

Installations- und ausführliche Verwendungshinweise: http://lehrerfortbildung-bw.de/werkstatt/sicherheit/stickcrypt/vc/

Schirmlizenzmodelle (MPLC) für privat erworbene Filme

Brauchen Schulen die regelmäßig von MPLC angebotenen Schirmlizenzmodelle für den Unterricht?

Hintergrund: Spielfilme, die in Videotheken und im Handel erhältlich sind, dürfen prinzipiell nur für private Zwecke genutzt werden, also beispielsweise im Wohnzimmer mit Freunden und Bekannten im kleinen Kreis. Unterricht wurde dagegen von der Film- und Verwertungsindustrie (wie z. B. MPLC) lange als nicht-privat und daher im juristischen Sinn als öffentlich angesehen.

Die von der MPLC Filmlizenzierung GmbH angebotenen Schirmlizenzmodelle mit jährlich fälligen Tarifen ("pauschale Nutzungslizenz") beziehen sich nur auf bestimmte, für die private Verwendung beschaffte Medien (d. h. nur die von Vertragspartnern von MPLC stammenden Filme, also nicht alle Spielfilme, die man privat aktuell kaufen kann).
Auch liegt dem Angebot die Auffassung zugrunde, Filmvorführungen im Unterricht seien juristisch gesehen öffentlich.

Das ZSL Baden-Württemberg vertritt auf seinem LehrerInnenfortbildungsserver allerdings ebenso wie das  Kultusministerium Baden-Württemberg eine gegenteilige Auffassung: Hier wird die Vorführung privater Filme im Klassenverband im Rahmen des als nicht-öffentlich aufgefassten Schulunterrichts als unproblematisch angesehen. (Dies gilt für Filme 2 Jahre nach dem Kinostart.)

Auch der Deutsche Landkreistag (DLT) vertritt diesen Standpunkt und die Einschätzung, dass der Abschluss entsprechender Lizenzvereinbarungen (wie die genannte Schirmlizenz) in der Regel nicht erforderlich sei: Nach der Rechtsauffassung der Bundesvereinigung der kommunalen Spitzenverbände sei "das Vorführen von Filmformaten (DVD etc.), die Lehrer oder Schulen im Handel erworben haben, urheberrechtlich jedenfalls dann zulässig, wenn es im Klassenverband erfolgt." (Rundschreiben des Landkreistags Baden-Württemberg Nr. 285/2013 vom 08.03.2013). "In Schulen ist die Besonderheit zu beachten, dass nach verbreiteter und auch von den kommunalen Spitzenverbänden geteilter Rechtsauffassung nicht von einer öffentlichen Wiedergabe auszugehen ist, soweit die Filme innerhalb eines Klassenverbands gezeigt werden." (Rundschreiben des Landkreistags Baden-Württemberg Nr. 486/2019 v. 16.4.2019)

Der Unterschied zwischen beiden Auffassungen ist m. W. juristisch nicht abschließend geklärt.

Was soll ich nach der Schule machen?

Schule endlich geschafft. Und jetzt? Vielleicht was mit Medien? Oder ein gap year? Da haben wir was für Dich: ein FSJ, bei dem Du Dein Interesse an digitalen Medien und neuen Technologien ausleben darfst. Das FSJ digital.

Du kannst bei uns ein Jahr lang mit spannenden digitalen Themen an Schulen und außerschulisch kreativ arbeiten! Dein Wissen an andere weitergeben, Dich mit Gleichgesinnten vernetzen und selbst eine Menge dazulernen. Danach weißt Du, was wirklich Dein Ding ist, und kannst in eine Ausbildung oder ins Studium starten.

Einsatzstelle finden
Was passiert beim FSJ digital?

Du bist aktiv bei Games, TikTok oder Podcasts dabei, liebst es zu coden oder interessierst Dich einfach für alles Digitale? Du möchtest Dein Know-how weitergeben, arbeitest gerne im Team und kannst gut erklären? Dann bist Du hier bei uns genau richtig! Spannende Projekte rund um digitale Medien warten auf Dich. Du arbeitest mit Kindern, Jugendlichen oder auch älteren Menschen. Du hilfst beim Lernen, Verstehen, Anwenden. Du tauschst Dich bei Seminaren mit anderen FSJlerinnen und FSJlern aus. Und vor allem: Du tust nach der Schule etwas Sinnvolles, das auch noch Spaß macht.

Makerspace
Die Makerspaces an den Medienzentren sind offene Werkstätten, in denen eigene Ideen, Prototypen oder Ersatzteile selbst hergestellt werden können. Mit 3D-Drucker, Laser-Cutter, Pressen oder Fräsen können dann faszinierende Projekte aus Holz, Stoff, Kunststoff oder Metall erarbeitet werden.
Robotik und Coding
Robotik und Coding
Fahren, sprechen, lachen, tanzen, blinken oder greifen – entdecke was mit verschiedensten Roboter-Typen möglich ist. Du erfährst, was das mit Logik, Abstraktion, sozialer Kompetenz und Teamfähigkeit zu tun hat und erweckst Roboter aus Deinem MediaCase zum Leben.
Podcast
Podcast
Wie entsteht eigentlich ein Podcast? Beim FSJ digital kommst Du vom Zuhören zum Produzieren. In spannenden Seminaren lernst Du im Team jede Menge über kreative Bildungsarbeit mit Medien. Du produzierst Beiträge zu Themen, die für Dich wichtig sind und andere unterstützen und weiterbringen.
Fotografie und Videoproduktion
Fotografie und Videoproduktion

Wer ist Träger des FSJ digital?

Das FSJ digital wird vom Landesmedienzentrum Baden-Württemberg (LMZ) angeboten. Das LMZ ist eine Landesinstitution mit Sitz in Karlsruhe und Stuttgart und bildet gemeinsam mit Stadt- und Kreismedienzentren an rund 50 Standorten im Land den Medienzentrenverbund. Es bietet pädagogische sowie technische Beratung und Weiterbildung, bildungsplanbezogene, fachlich begutachtete Medien sowie Support für schulische Computernetzwerke.

Warum ein FSJ digital?
Du weißt noch nicht, was Du nach der Schule machen sollst – suchst aber eine sinnvolle Aufgabe, die richtig Spaß macht. Und Du möchtest Dich persönlich weiterentwickeln.
Medien und Kommunikation sind Dein Ding, und Du willst ausprobieren, was man damit alles machen kann. Du suchst Ideen für Deine Berufswahl.
Du bist neugierig und kreativ und willst eigene Ideen gemeinsam mit anderen umsetzen. Du möchtest Dich mit Gleichgesinnten vernetzen und suchst eine Community mit gleichen Interessen.
Du liebst es, mit anderen im Team zu arbeiten, kannst Dinge gut erklären und gibst Dein Wissen gerne in Schulen und an außerschulischen Lernorten weiter.

WER MACHT MEIN-HEILBRONN.ORG?

Dieses Projekt haben sich das Stadtarchiv, das Kreismedienzentrum und das Staatliche Schulamt Heilbronn für die Schulkinder aus Heilbronn und Umgebung ausgedacht.

Zusammen mit vielen Helferinnen und Helfern, und mit mehreren Autorinnen und Autoren, ist dieses umfangreiche und vielseitige Angebot entstanden.

Mein-Heilbronn.org wird von der Bürgerstiftung Heilbronn großzügig gefördert. Die Firma Baier & Schneider (Brunnen) hat die erste Auflage von Mein-Heilbronn-Blanco-Heften zur individuellen Gestaltung sowie Klassenzimmer-Poster kostenlos zur Verfügung gestellt.

Die Projektleitung, Redaktion und Weiterentwicklung hat das Kreismedienzentrum Heilbronn übernommen.

Die Gesamtleitung und Sachfinanzierung liegt beim Stadtarchiv Heilbronn.

Was ist Ohrenspitzer?

Das Projekt "Ohrenspitzer" der Stiftung MedienKompetenz Forum Südwest (MKFS) fördert das bewusste Hören als Grundlage gelingender Kommunikation und stellt zahlreiche Methoden der Zuhörförderung bereit.

Zielgruppe sind in erster Linie Grundschulkinder sowie Kinder an Kindertagesstätten ("Ohrenspitzer-mini") . Es gibt aber auch Hörangebote für Kinder ab 10 J.

Für Lehrkräfte und ErzieherInnnen gibt es ein Workshop-, Fortbildungs- und Beratungsangebot, das von qualifizierten Ohrenspitzer-ReferentInnen durchgeführt wird.

Mein-Heilbronn.org - Entdecke deine Stadt
interaktives Heimatbuch

Mein-Heilbronn.org begleitet Schülerinnen und Schüler (ab Klasse 3) dabei, Heilbronn mit seinen Stadtteilen und seiner Geschichte anhand dilgitaler bzw. analoger Aufgaben und Medien sowohl im Klassenzimmer als auch vor Ort kennen zu lernen. Dabei gestalten sie ihr eigenes Heilbronn-Heft oder -Buch. 

Mein-Heilbronn.org
allgemeine Themen

Auch für Schulen des Landkreises interessant. Viele allgemeine Themen sind am Beispiel Heilbronn dargestellt: die politische Gliederung Baden-Württembergs, Müllentsorgung, Feuerwehr, uvm.

Mein-Heilbronn.org
das Projekt

Vorläufer des Projekts war ein viele Jahre bewährtes Druckwerk namens „Wir wohnen in Heilbronn – Kinder lernen ihre Stadt kennen“ aus dem Jahre 2003. Nach einer letzten Überarbeitung 2008 entschied sich die Stadt Heilbronn, die Inhalte im interaktiven Online-Format auflegen zu lassen. Am 27.1.2015 wurde die Plattform offiziell in Betrieb genommen und bis 2023 redaktionell gepflegt.

Foto: (v. l.): Karl Schäuble, Vorsitzender Bürgerstiftung Heilbronn; Bürgermeisterin Agnes Christner; Prof. Dr. Christhard Schrenk, Leiter Stadtarchiv Heilbronn; Folkhard Leuschner, Leiter Kreismedienzentrum Heilbronn; Wolfgang Seibold, Leiter Staatliches Schulamt Heilbronn

iPad-Verwaltung per MDM
MDM für Schulen mit Tablets – Vollservice empfohlen

Schulen oder Schulträgern, denen die notwendigen zeitlichen und personellen Ressourcen für die eigene Administration ihrer iPads nicht zur Verfügung stehen, empfehlen wir, entsprechende kostenpflichtige Dienstleistungsangebote von externen zertifizierten Dienstleistern (Apple Education Specialists) zu beauftragen, die auf die Verwaltung von iPads für Schulträger und Schulen spezialisiert sind.

Wünschenswert aus Sicht der Schulen wäre dabei:

An der Schule ist lediglich eine Koordinationsperson notwendig (kein Admin). Diese übermittelt die Wünsche bezüglich der App-Ausstattung, Einstellungen etc. aus Sicht der Benutzenden.
Es können sowohl vorhandene Geräte integriert als auch neue Geräte, auch aus anderen Quellen, verwaltet werden. Wichtig dabei ist die DEP-Fähigkeit.
Vielfach bieten Schulträger, wo es möglich ist, an, die ASM- sowie die MDM-Instanz zentral über den Träger laufen zu lassen, und die zugeordneten Schulen als sog. Standorte. Dies hat Vorteile bei der Geräteverwaltung und bei der App-Beschaffung und -Verteilung.

Als technische Grundlage ist ein geeignetes MDM (wie z. B. Jamf School - vormals Zuludesk) zu wählen. MDM steht für Mobile Device Management.

Geräte und App-Lizenzen werden über den Apple School Manager (ASM) verwaltet, und damit wirtschaftlich unabhängig vom MDM-Dienstleister.

Für Vergleichsangebote für die MDM-Verwaltungs-Dienstleistung (oder auch Gerätebeschaffung) finden Sie die Liste der zertifizierten Apple Education Specialists (vormals: Solution Experts) auf https://www.apple.com/de/education/how-to-buy/education-specialist/ 

Darüber hinaus gibt es natürlich zahlreiche weitere qualifizierte Firmen und Dienstleister, die iPads und/oder deren Verwaltung per MDM anbieten. Ein Preis- und Leistungsvergleich ist also möglich und sinnvoll.

iPads im Unterrichtseinsatz
Unterrichtsgestaltung mit iPad

iPads können gut als Dokumentenkamera genutzt werden. Dazu wird ein Beamer bzw. Monitor (ggf. mit AppleTV) benötigt.
Verwendet werden iPad-Halterungen wie z. B. "Belkin table stage" oder der "Value Tischhalter für Apple iPad/Ebook/Tablet-PC". Diese Halterungen lassen sich in alle möglichen Richtungen drehen und in verschiedene Winkel einstellen. Dadurch können sie auch als Stativ für Filmaufnahmen verwendet werden (z. B. auch zum Trickfilmen).

Zu Hause vorbereitete Unterrichtsinhalte können den SuS präsentiert werden. Eine Einbindung ins W-LAN ist sinnvoll, um Zugriff auf Internetinhalte zu haben (z. B. didaktische Medien bei SESAM oder Planet Schule).

Medienproduktion mit dem iPad

Einige Möglichkeiten für praktische Medienarbeit sind: Trickfilm (z.B. mit iStopMotion), Videoschnitt (z.B. mit iMovie), Musik und Ton (z.B. mit Garageband, Quickvoice, Lesespurgeschichten), Präsentieren (z.B. mit Keynote, Book Creator), E-Book-Authoring (z.B. mit Book Creator).

Auch Internetrecherche, Textverarbeitung und Tabellenkalkulation und der Einsatz von Lernprogrammen sind natürlich auch mit Tablets möglich. Eine sinnvolle Ergänzung dabei sind eine Tastatur, z. B. "Logitech Slim Folio" und ein Apple Pencil. Je nach Einsatz kann eine Display-Folie, die die Haptik des Schreibens auf Papier vermittelt, sinnvoll sein.

iPad-Verwaltung

Grundsätzlich raten wir bei der Anschaffung von schuleigenen iPads zur Verwaltung über ein Mobile Device Management (MDM). Diese Administration sollte als Serviceleistung Schulen zur Verfügung gestellt werden, entweder vom Träger oder von einer externen Firma.

Moodle
Lern- und Lehrplattform für alle Schulen

Das für Schulen in Baden-Württemberg angebotene Moodle incl. Videokonferenztool Big Blue Button (BBB) wird inhaltlich vom Zentrum für Schulqualität und Lehrerbildung (ZSL) betreut und fortgebildet.

SESAM-Einbindung ins Schul-Moodle

Lehrkräfte, die die Lernplattform Moodle als Admins verwalten, können jetzt eine „Direktanbindung“ zur SESAM-Mediathek über einen LTI-Schlüssel global für die ganze Schule konfigurieren. Bei der globalen Anbindung von SESAM an Ihr Schul-Moodle ist nur eine einmalige Konfiguration notwendig, damit alle berechtigten Lehrkräfte kursübergreifend auf diese LTI-Anbindung zugreifen können. So können alle Kolleginnen und Kollegen direkt aus Moodle heraus in SESAM nach Medien suchen und diese in ihren Kurs einbinden. Natürlich muss die Verwendung dieser Konfiguration durch die Nutzenden des Schul-Moodle entsprechend der Allgemeinen Nutzungsbedingungen von SESAM erfolgen. Sie darf also ausschließlich den Lehrkräften der Schule zugänglich gemacht werden. 

Eine genaue Anleitung zur globalen Konfiguration der SESAM-Anbindung in Moodle via LTI finden Sie im PDF LTI-Schluessel-globale-Einbindung.pdf sowie im Hilfebereich von SESAM.

HINWEIS: Moodle bei BelWue

Die Moodle-Instanzen von Schulen bei BelWue werden mittelfristig zu einem anderen Dienstleister umgezogen. Es besteht allerdings kein Handlungsdruck.

Zitat aus der Mail des BelWue vom 30. April 2021 an alle Schulen "Die Dienste „E-Mail“ und Lernplattform „Moodle“ bleiben vorerst unverändert bestehen. Mittelfristig sollen diese von einem anderen zentralen  Dienstleister übernommen werden.
